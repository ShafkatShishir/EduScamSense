# EduScamSense

EduScamSense is a bilingual Bangla–English cybersecurity education platform designed to help educators ( Teachers ) recognize and respond safely to phishing, voice-cloning, and deepfake scam scenarios.

The project combines a lightweight machine-learning phishing-text classifier, expert-reviewed educational scenarios, interactive awareness modules, quizzes, and practical verification guidance. The platform is developed using Python, scikit-learn, and Gradio, with deployment planned through Hugging Face Spaces.

EduScamSense is intended for cybersecurity education and awareness. It does not directly detect cloned audio or manipulated video and should not replace professional cybersecurity procedures or independent verification.
